# Task 094: Micro-interações, Transições e Polimento Visual

## Objetivo
Tornar o sistema visualmente mais vivo e polido, adicionando feedbacks táteis de interface.

## Subtarefas
- [ ] Instalar `framer-motion` (se ainda não presente).
- [ ] Criar animação de "Page Transition" (Fade-in/Up) ao navegar entre rotas.
- [ ] Adicionar efeito "Scale-down" (0.95-0.98) ao clicar em botões principais.
- [ ] Implementar animações de feedback em `StatusBadge` (quando o status muda).
- [ ] Adicionar animações suaves ao abrir e fechar a Sidebar (se ainda não fluida).
- [ ] Criar micro-animação para o Command Palette (abertura centralizada e suave).

## Definição de Pronto (DoP)
- Sistema se sente "responsivo" e "premium" através de feedbacks táteis.
- Mudanças de tela não são abruptas.
- Animações respeitam a configuração de "reduzir movimento" do sistema operacional.
