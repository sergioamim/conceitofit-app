# Task 093: Audit de Acessibilidade, Contraste e Estados de Foco

## Objetivo
Garantir que a aplicação seja acessível e tenha uma navegação via teclado clara e vibrante.

## Subtarefas
- [ ] Realizar audit de contraste da cor `gym-accent` (#c8f135) com texto branco e fundos escuros.
- [ ] Implementar variante de "Contraste Seguro" para textos pequenos.
- [ ] Customizar o anel de foco global (`focus-visible`) no Tailwind para usar a cor da marca com brilho (glow).
- [ ] Adicionar suporte a `skip-to-content` (link invisível para saltar a sidebar).
- [ ] Testar navegação completa via teclado (Tab/Enter/Space) na Sidebar e Tabelas.

## Definição de Pronto (DoP)
- Todos os elementos interativos têm foco visualmente claro.
- Contraste de texto respeita as normas de acessibilidade (AA).
- Navegação fluida sem o uso de mouse.
