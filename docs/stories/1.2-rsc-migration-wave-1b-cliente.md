## Contexto

Após a migração bem-sucedida da Wave 1 (Relatórios Contábeis + Catálogo Segurança), aplicamos o mesmo padrão **Server Component + Client Islands** para a **área do aluno** do academia-app.

Embora essas páginas sejam menos complexas que as do backoffice, elas representam o **maior volume de tráfego** do app — todos os alunos acessam diariamente. Qualquer melhoria de performance impacta milhares de usuários.

### Páginas alvo

| Página | Linhas | Padrão Atual | Ganho Esperado |
|--------|--------|-------------|----------------|
| `cliente/meu-perfil/page.tsx` | 167 | `useClienteOperationalContext` (React Query) | ~30% TTI |
| `cliente/check-in/page.tsx` | 325 | `useEffect` + fetch client | ~40% TTI |
| `cliente/indicar/page.tsx` | 322 | `useEffect` + fetch client | ~40% TTI |
| `cliente/minhas-avaliacoes/page.tsx` | 452 | Client Component monolítico | ~35% TTI |
| `cliente/loja/page.tsx` | 493 | `useEffect` + fetch client | ~40% TTI |

### Já migradas (não precisam de ação)
- `cliente/minhas-aulas/page.tsx` → delega para `MinhasAulasClient` (5 linhas)
- `cliente/meus-treinos/page.tsx` → delega para `MeusTreinosClient` (5 linhas)
- `cliente/meus-pagamentos/page.tsx` → delega para Client (5 linhas)
- `cliente/meus-contratos/page.tsx` → delega para Client (5 linhas)

---

## Objetivo

Migrar **5 páginas da área do cliente** para o padrão **Server Component + Client Islands**.

**Ganho esperado:**
- TTI reduz **30-40%** nessas páginas
- Bundle inicial do browser reduz **~20-40KB** por página
- Impacto direto em **todos os alunos** da plataforma

## Pattern de Referência

Mesmo padrão da Wave 1 (`leads/page.tsx` e `relatorios/page.tsx`):

```tsx
// page.tsx — Server Component
import { serverFetch } from "@/lib/shared/server-fetch";
import { MeuPerfilContent } from "./components/meu-perfil-content";

export const dynamic = "force-dynamic";

async function Loader() {
  const data = await serverFetch("/api/v1/cliente/operational-context", { next: { revalidate: 0 } });
  return <MeuPerfilContent initialData={data} />;
}

export default function Page() {
  return <Suspense fallback={<Loading />}><Loader /></Suspense>;
}
```
